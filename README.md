# BuboCore

[![Crates.io](https://img.shields.io/crates/v/bubocore.svg)](https://crates.io/crates/bubocore) [![Documentation](https://docs.rs/bubocore/badge.svg)](https://docs.rs/bubocore) [![Build Status](https://github.com/username/bubocore/workflows/Rust/badge.svg)](https://github.com/username/bubocore/actions)

BuboCore est un séquenceur et un environnement de programmation musicale programmé en Rust. BuboCore est conçu pour encourager la pratique du [live coding](https://livecoding.fr) musical.

## Documentation

Pour la documentation, dirigez vous sur [bubocore.livecoding.fr](https://bubocore.livecoding.fr).

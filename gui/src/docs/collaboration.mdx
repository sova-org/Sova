# Collaboration

Sova supports real-time collaboration for live coding performances.

## Real-time Features

### Peer Awareness
See other users' cursors and selections in real-time:
- **Cursor Tracking**: See where other users are typing
- **Selection Highlighting**: View other users' selected content
- **User Colors**: Each collaborator has a unique color

### Synchronized Playback
All connected clients share the same playback state:
- **Transport Sync**: Play/stop affects all users
- **Tempo Sync**: Tempo changes are shared
- **Position Sync**: Playback position is synchronized

### Live Updates
Changes propagate immediately across all connected clients:
- **Script Updates**: Code changes sync in real-time
- **Grid Changes**: Frame modifications are shared
- **Language Changes**: Language selection is synchronized

## Connection Management

### Server Connection
- **Status Display**: Connection status shown in footer
- **Auto-reconnect**: Automatic reconnection on connection loss
- **Error Handling**: Graceful handling of network issues

### User Management
- **Username**: Click your name in the footer to change it
- **Peer Count**: See how many users are connected
- **Server Address**: View current server connection

## Best Practices

### Performance Tips
- **Coordinate Changes**: Communicate before making major changes
- **Use Comments**: Document your code for other collaborators
- **Test Individually**: Test your code before sharing

### Communication
- **Voice Chat**: Use external voice chat for coordination
- **Screen Sharing**: Share your screen for complex explanations
- **Code Comments**: Use comments for in-code communication

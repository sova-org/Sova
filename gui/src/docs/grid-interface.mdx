# Grid Interface

The grid provides a visual timeline for organizing and triggering musical sequences.

## Frame Management

### Selection
- **Click**: Select a single frame
- **Shift+Click**: Select multiple frames
- **Drag**: Select a range of frames

### Editing
- **Double-click**: Edit frame script
- **Right-click**: Access context menu
- **Drag**: Move frames within the timeline

### Creation
- **Insert**: Add new frames
- **Delete**: Remove selected frames
- **Copy/Paste**: Duplicate frame content

## Visual Indicators

### Language Colors
Each programming language has a distinct color:
- **Bali**: Blue tones
- **Boinx**: Green tones  
- **DummyLang**: Orange tones

### Compilation Status
- **Green Border**: Successfully compiled
- **Red Border**: Compilation errors
- **Gray Border**: Not compiled

### Playback Position
- **Highlighted Frame**: Current playback position
- **Dimmed Frames**: Past playback positions

## Timeline Navigation

### Zoom Controls
- **Scroll**: Zoom in/out of timeline
- **Ctrl+Scroll**: Horizontal scroll
- **Shift+Scroll**: Vertical scroll

### Playback Following
The grid automatically follows playback position during performance.

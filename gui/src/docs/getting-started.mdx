# Getting Started

**Sova** is a powerful live coding environment for musical performance and composition. 
Connect to a Sova server to start creating music with code.

## Quick Start

Click the "Connect" button to connect to a local server, or use the splash screen to connect to a remote server.

## First Steps

1. **Connect to Server**: Use the connection dialog to connect to a Sova server
2. **Choose Your View**: Switch between Editor, Grid, or Split view using the view buttons
3. **Start Coding**: Begin writing musical code in the editor
4. **Play Your Music**: Use the play button to hear your creations

## System Requirements

- Modern web browser with JavaScript enabled
- Access to a Sova server (local or remote)
- Audio output device for playback

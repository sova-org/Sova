# Tips & Tricks

Master Sova with these helpful tips and advanced techniques.

## Keyboard Shortcuts

### Global Shortcuts
- **Ctrl+K**: Open command palette for quick access to all features
- **Ctrl+Enter**: Evaluate current script
- **Ctrl+,**: Open options panel

### View Navigation
- **Ctrl+1**: Switch to Editor view
- **Ctrl+2**: Switch to Grid view
- **Ctrl+3**: Switch to Split view
- **Ctrl+?**: Open this help documentation

## Advanced Features

### Command Palette
Access all functionality through the command palette:
- **Quick Actions**: Execute any menu action
- **View Switching**: Jump between different views
- **Settings**: Access configuration options

### Grid Interaction
- **Right-click**: Context menus for frame operations
- **Drag Selection**: Select multiple frames efficiently
- **Copy/Paste**: Duplicate successful patterns

### Code Editor
- **Language Selection**: Switch languages per frame
- **Error Navigation**: Click errors to jump to problem lines

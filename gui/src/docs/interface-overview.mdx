# Interface Overview

Sova's interface is designed for efficient live coding and musical performance.

## View Modes

### Editor View
Focus on writing code with a full-screen editor experience. Perfect for detailed script development.

### Grid View
Visual timeline view for arranging musical sequences. Ideal for organizing and triggering patterns.

### Split View
Combined editor and grid view with resizable panels. The best of both worlds for live performance.

## Playback Controls

### Transport Controls
- **Play Button**: Start playback of your musical sequences
- **Stop Button**: Stop current playback
- **Tempo Control**: Interactive tempo adjustment

### Tempo Bar
The tempo bar shows the current playback position and allows real-time tempo adjustments:
- **Left Click**: Decrease tempo by 1 BPM
- **Right Click**: Increase tempo by 1 BPM  
- **Shift + Click**: Adjust by 5 BPM increments

### Phase Indicator
Visual feedback showing the current playback position within the musical phrase.

The interface is composed of a different "views" organised around a central navigation grid. Navigation is heavily based on keyboard shortcuts for speed and efficiency.

## Controls

*   `TAB` - Open/Close the navigation menu

### In Navigation Grid:

*   `↑` / `↓` / `←` / `→` : Move cursor on the map.
*   `A-Z`: Quickly jump to view by name (letter).

### General:

*   `F1` - Switch to **Editor** view
*   `F2` - Switch to **Grid** view
*   `F3` - Switch to **Options** view
*   `F4` - Switch to **Help** view
*   `F5` - Switch to **Devices** view
*   `F6` - Switch to **Logs** view
*   `F7` - Switch to **Files** view
*   `Ctrl+P` - Open command prompt
*   `Ctrl+C` - Exit application

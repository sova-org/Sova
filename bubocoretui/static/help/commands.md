Type `Ctrl+P` to open the command prompt, then enter commands:

## General

*   `quit` / `q` / `exit` / `kill` - Exit the application
*   `connect` - Attempt to reconnect to the server

## Views

*   `editor` - Switch to Editor view
*   `grid` - Switch to Grid view
*   `options` - Switch to Options view
*   `devices` - Switch to Devices view
*   `logs` - Switch to Logs view
*   `help` / `?` - Show this help screen
*   `files` - Switch to save/load view

## Network

*   `setname [name]` - Change your client name
*   `chat [message]` - Send a chat message to other clients

## Clock

*   `tempo [bpm]` / `t [bpm]` - Set tempo (20-999 BPM)
*   `quantum [beats]` - Set Link quantum/measure length (0-16 beats)

Unknown commands will be logged as errors.

The `Files` view allows you to save and load snapshots of the scene currently playing.

## Keybindings

### List View Mode

*   `↑` / `↓` : Navigate Project List
*   `/` : Filter project list (search)
*   `l` : Load Project (immediately, without transition)
*   `Ctrl` + `L` : Load Project (at end of current scene)
*   `Ctrl` + `D` : Delete Selected Project
*   `s` : Save Project

### Saving Mode

*   `Enter` : Confirm Save
*   `Esc` : Cancel Save
*   *(Type)* : Enter project name

### Searching/Filtering Mode

*   `Esc` : Clear search & Exit Mode
*   `Enter` : Keep filter & Exit Mode
*   `Backspace` : Delete character
*   *(Type)* : Update search query

    Installation

    • Pour les développeurs
    • Pour les utilisateurs

    Pour les développeurs #

    1. Cloner le dépôt logiciel de BuboCore: git clone https://github.com/Bubobubobubobubo/bubocore. La branche main représente toujours la version stable la plus récente.
    2. Depuis la ligne de commande, exécuter : cd bubocore && cargo run build.

    Pour les utilisateurs #

    Installation #

    1. cargo install bubocore
    2. Profit.
